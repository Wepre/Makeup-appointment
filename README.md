# 基于云开发的化妆品预约系统设计与实现 云开发小程序 后台采用cms内容管理系统 实现了对化妆品到店预约功能 不适合上线 功能整体比较全 有视频 可以联系我看看

## 主要功能如下

# 小程序端

- 用户注册登录
- 用户信息修改
- 论坛发布
- 分类查看
- 商品查看
- 商品预订
- 商品预订列表
- 确认收货
- 费用支付
- 订单完成
- 商品发布评论
- 预订中心（相当于购物车）
- 选择地址.
- 收藏商品
- 用户反馈
- 
  

# 后台端
- 化妆品管理
- 分类管理
- 订单管理
- 评论管理
- 用户管理
- 轮播图管理
- 收货地址管理
- 反馈管理
